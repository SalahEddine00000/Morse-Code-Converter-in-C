# Morse Code Converter in C

A terminal-based program to **convert text ↔ Morse code**.

## Features
- Morse → Text conversion
- Text → Morse conversion
- Display Morse table
- Clear screen option
##Menu
-0 → Exit
-1 → Morse to Text
-2 → Text to Morse
-3 → Display Morse Table
-4 → Clear Screen



## Compilation & Run
```bash
gcc main.c -o morse
./morse
